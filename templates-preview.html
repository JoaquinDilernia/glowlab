📖 ¿Qué son los Templates Predeterminados?

Cuando una tienda instala PromoNube por primera vez, el sistema crea automáticamente 4 diseños profesionales listos para usar. Estos templates:

  • Se instalan sin intervención del usuario
  • Cubren diferentes estilos visuales (elegante, premium, suave, minimalista)
  • Están optimizados para diferentes tipos de marcas
  • Incluyen configuración de texto predefinida (color, tamaño, posición)
  • El primero se marca automáticamente como "default"

🔧 Implementación Técnica

Función: installDefaultTemplates(storeId)

Cuándo se ejecuta: Automáticamente en el auth callback después de registrar la tienda

Almacenamiento: SVG en formato base64 dentro del data URL

Colección Firestore: giftcard_templates

Campo especial: isSystemTemplate: true para diferenciarlos de templates custom
